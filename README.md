# AlbsigBay - Online Bücherleihe
# Installation
## 1. First clone the repo:
```bash
git clone https://github.com/Blueblazer172/albsigbay.git

cd albsigbay
```
## 2. Install Dependencies
```bash
npm ci
```
# Run
```bash
node server.js
```
and
```bash
node api.js
```
